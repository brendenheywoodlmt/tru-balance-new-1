Tru Balance | Professional Massage & Bodywork
A custom, high-performance web platform built for Tru Balance, a massage therapy practice led by a high-reach industry influencer. This project involved a complete architectural refactor to replace an incomplete legacy build, ensuring the digital presence matched the professional standards of the brand.

🛠 Project Overview: The "Rescue" Mission
The client moved from Wix to a custom solution to gain full ownership of their digital assets. After an initial developer delivered an incomplete site with broken styling and inaccurate service data, I performed a full-stack overhaul to stabilize the platform and align it with the client’s meticulous design requirements.

Key Deliverables:
Complete Code Refactor: Audited and rewrote the existing codebase to fix broken CSS/color schemes and eliminate "ghost" services left by the previous developer.

Square Integration: Synchronized all "Book Now" actions with the client’s Square Appointments backend for seamless scheduling.

Influencer Branding: Translated a sophisticated logo and color palette (designed by professional graphic designers) into a modern, "spa-like" digital aesthetic using Tailwind CSS.

💻 Tech Stack
Framework: Next.js / React

Styling: Tailwind CSS (featuring custom hex-code implementation for brand-accurate consistency)

Deployment: Vercel

Development Tools: Gemini (Syntax optimization), VS Code Intellisense (Refactoring efficiency)

✨ Core Features
Service Accuracy: A cleaned and verified service menu reflecting real-time offerings and pricing, removing the clutter of inaccurate legacy data.

Patient Education Hub: A dedicated Resources Page where the client—a therapist with over a decade of experience—shares insights, debunks industry myths (e.g., Soft vs. Deep Tissue), and provides patient-first education.

SEO Foundations: Implemented technical SEO best practices, including JSON-LD schema and automated sitemaps, to improve local search visibility for massage services.

Mobile-Optimized UX: Designed specifically for a social-media-heavy audience, ensuring the site performs flawlessly when accessed via links from Instagram, TikTok, or YouTube.

📈 Business Impact
By moving the site to a custom Next.js environment, the client now has a platform that:

Reflects Brand Authority: Matches the visual quality expected by his thousands of social media followers.

Reduces Admin Friction: Direct Square integration ensures that "viral" traffic converts into scheduled appointments without manual intervention.

Future-Proofed: Provided the client with documentation on local development environments (npm run dev) and deployment workflows to ensure long-term stability.
